사회재난 저감성 평가

  • 저감성 지표코드 (24개, SV1~36) 데이터 수집

    재난별로 활용되는 지표코드 상이

    ex）저감성화재 :
    의사 수(SR16), 의료인력 수(SR16), 특수의료장비 현황(SR17), 의료기관 수(SR14), 공무원 수_정원(SR5), 구조구급대원 수(SR7), 소방공무원 수(SR9)
  • 저감성 지표 data : 지자체별로 지표코드 raw data 값 정리
  • 저감성 mapping (표준화) : (저감성지표 data값 – 최소값) / (최대값 – 최소값)
  • 저감성 (요인분석지표) : 재난별 저감성 mapping 결과값 평균

자연재난 저감성 평가

  • 저감성 지표명 (15개) 데이터 수집

    재난별로 활용되는 지표명 상이

    ex）저감성 태풍/호우/강풍 :
    의료인력 수, 의사 수, 특수의료장비현황, 의료기관 수, 구조구급대원 수, 공무원 수(정원), 소방공무원 수, 내수침수 재해 행정관리능력, 하천재해 행정관리능력, 자연재해 위험개선지구 정비, 방재 유관기관 네트워크 구축, 재난상황대응
  • 저감성 지표 data : 지자체별로 지표코드 raw data 값 정리
  • 저감성 mapping (표준화) : (저감성지표 data값 – 최소값) / (최대값 – 최소값)
  • 저감성 (요인분석지표) : 재난별 저감성 mapping 결과값 평균
